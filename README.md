# Unit-Converter

A simple unit converter built with HTML, CSS, and JavaScript. This project allows users to quickly convert values between different units directly in the browser.

---

## Built with

- HTML
- CSS
- Javascript (Vanilla)
- Netlify

---


## Features

- Convert values between different units
- Button-triggered conversion
- Clean and minimal UI
- Lightweight — runs entirely in the browser


---

**What I learned in this section:**

- Writing conversion logic using JavaScript functions
- Working with user input and form elements
- Updating the DOM dynamically with results
- Handling different unit types and calculations
- Structuring a small interactive web application
